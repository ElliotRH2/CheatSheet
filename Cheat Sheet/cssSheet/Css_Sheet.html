CSS Cheat Sheet

Selectors

1. **Universal Selector:**
- `*`: Selects all elements on the page.

2. **Type Selector:**
- `element`: Selects all elements of a specific type (e.g., `div`, `p`, `h1`, etc.).

3. **Class Selector:**
- `.classname`: Selects all elements with the specified class name (e.g., `.container`, `.button`, etc.).

4. **ID Selector:**
- `#idname`: Selects a single element with the specified ID (IDs must be unique) (e.g., `#header`, `#main`, etc.).

5. **Descendant Selector:**
- `ancestor descendant`: Selects all descendants of the ancestor element (e.g., `ul li`, selects all `li` elements that are descendants of `ul` elements).

6. **Child Selector:**
- `parent > child`: Selects all direct children of the parent element (e.g., `ul > li`, selects all `li` elements that are direct children of `ul` elements).

7. **Adjacent Sibling Selector:**
- `element + sibling`: Selects an element that is immediately preceded by a specified sibling element (e.g., `h2 + p`, selects all `p` elements that are immediately preceded by `h2` elements).

8. **General Sibling Selector:**
- `element ~ sibling`: Selects all sibling elements that follow a specified element (e.g., `h2 ~ p`, selects all `p` elements that follow `h2` elements).

9. **Attribute Selector:**
- `[attribute]`: Selects all elements with the specified attribute (e.g., `[type]`, selects all elements with a `type` attribute).
- `[attribute=value]`: Selects elements with the specified attribute and value (e.g., `[type=text]`, selects all elements with `type` attribute equal to `text`).

10. **Pseudo-class Selector:**
- `:hover`: Selects an element when you hover over it with the mouse.
- `:active`: Selects an element when it is being activated (e.g., clicked).
- `:focus`: Selects an element when it has focus.
- `:nth-child(n)`: Selects elements based on their position within a parent element.
- `:first-child`: Selects the first child element of its parent.
- `:last-child`: Selects the last child element of its parent.

CSS Properties

1. **Font Properties:**
- `font-family`: Sets the font family for text.
- `font-size`: Sets the size of the font.
- `font-weight`: Sets the boldness of the font.
- `font-style`: Sets the style of the font (italic, oblique, etc.).

2. **Color and Background Properties:**
- `color`: Sets the text color.
- `background-color`: Sets the background color.
- `opacity`: Sets the opacity level of an element.

3. **Layout Properties:**
- `width`: Sets the width of an element.
- `height`: Sets the height of an element.
- `margin`: Sets the margin around an element.
- `padding`: Sets the padding inside an element.
- `display`: Sets how an element is displayed (block, inline, etc.).

4. **Border and Outline Properties:**
- `border`: Sets the border around an element.
- `border-radius`: Sets the radius of the border corners.
- `outline`: Sets an outline around an element.

5. **Positioning Properties:**
- `position`: Sets the positioning method of an element (relative, absolute, fixed, etc.).
- `top`, `right`, `bottom`, `left`: Sets the position of an element relative to its containing element.
- `z-index`: Sets the stack order of an element.

6. **Text Properties:**
- `text-align`: Sets the horizontal alignment of text.
- `text-decoration`: Sets the decoration of text (underline, line-through, etc.).
- `line-height`: Sets the height of a line of text.
- `word-spacing`: Sets the spacing between words.
- `letter-spacing`: Sets the spacing between characters.

7. **Box Model Properties:**
- `box-sizing`: Sets how the total width and height of an element are calculated.
- `overflow`: Sets what to do when an element's content is too big to fit in its box.
- `visibility`: Sets the visibility of an element (hidden, visible, etc.).

8. **Flexbox Properties:**
- `flex-direction`: Sets the direction of the main axis in a flex container.
- `justify-content`: Sets how flex items are aligned along the main axis.
- `align-items`: Sets how flex items are aligned along the cross axis.
- `flex-grow`, `flex-shrink`, `flex-basis`: Sets the flex grow, shrink, and basis of a flex item.

9. **Grid Properties:**
- `grid-template-columns`, `grid-template-rows`: Sets the size of grid tracks in a grid container.
- `grid-column`, `grid-row`: Sets the start and end positions of a grid item.
- `grid-gap`: Sets the size of the gap between grid rows and columns.

10. **Animation and Transition Properties:**
- `animation`: Sets the keyframe-based animations to be applied to an element.
- `transition`: Sets the transition effects to be applied when a CSS property changes.

Media Queries

1. **Screen Width:**
- `@media screen and (max-width: 768px) { ... }`: Applies styles when the screen width is 768 pixels or less.
- `@media screen and (min-width: 1024px) { ... }`: Applies styles when the screen width is 1024 pixels or more.

2. **Device Orientation:**
- `@media screen and (orientation: landscape) { ... }`: Applies styles when the device is in landscape orientation.
- `@media screen and (orientation: portrait) { ... }`: Applies styles when the device is in portrait orientation.

3. **Device Resolution:**
- `@media screen and (min-resolution: 300dpi) { ... }`: Applies styles when the device has a minimum resolution of 300 dots per inch.
- `@media screen and (max-resolution: 96dpi) { ... }`: Applies styles when the device has a maximum resolution of 96 dots per inch.

4. **Viewport Width:**
- `@media (max-width: 600px) { ... }`: Applies styles when the viewport width is 600 pixels or less.
- `@media (min-width: 768px) { ... }`: Applies styles when the viewport width is 768 pixels or more.

5. **Device Type:**
- `@media handheld { ... }`: Applies styles when the device is a handheld device (e.g., mobile phones).
- `@media print { ... }`: Applies styles when the document is being printed.

6. **Aspect Ratio:**
- `@media screen and (aspect-ratio: 16/9) { ... }`: Applies styles when the aspect ratio is 16:9.
- `@media screen and (aspect-ratio: 4/3) { ... }`: Applies styles when the aspect ratio is 4:3.

7. **Combining Queries:**
- `@media screen and (min-width: 768px) and (max-width: 1024px) { ... }`: Applies styles when the screen width is between 768 and 1024 pixels.
- `@media screen and (orientation: landscape) and (max-width: 768px) { ... }`: Applies styles when the screen width is 768 pixels or less and the device is in landscape orientation.
