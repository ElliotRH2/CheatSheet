<html> Element

The <html> tag represents the root of an HTML document, and is the container for all other HTML elements (except for the <!DOCTYPE> tag).

Note:
You should always include the lang attribute inside
the <html> tag, to declare the language of the Web page.
This is meant to assist search engines and browsers.

<!--...--> Comment Tag

         

            

               In HTML, comments can be added between an opening <!-- and closing -->. Content inside of comments will not be rendered by browsers, and are usually used to describe a part of code or provide other details.
               Comments can span single or multiple lines.
